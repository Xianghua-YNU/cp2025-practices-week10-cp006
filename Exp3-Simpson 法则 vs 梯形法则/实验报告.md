# 数值积分方法比较实验报告（Simpson 法则 vs 梯形法则）

## 一、实验目的

1. 理解并掌握 Simpson 法则与梯形法则的数值积分原理及实现方法。
2. 通过实验比较两种方法在积分精度和误差方面的表现。
3. 能够分析不同子区间数 $N$ 对积分结果的影响。

## 二、实验原理

### 2.1 Simpson 法则

简述 Simpson 法则的积分思想、公式推导及适用条件（$N$ 必须为偶数），并给出公式：
$$
\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i=1,3,5,...}^{N-1}f(x_i) + 2\sum_{i=2,4,6,...}^{N-2}f(x_i) + f(x_N)\right]
$$
1. **积分思想**
   - Simpson 法则的基本思想是用二次函数（抛物线）来近似代替被积函数 \(y = f(x)\) 在每个子区间上的曲线段。
   - 将积分区间 \([a,b]\) 分成 \(N\) 个等宽的子区间，每个子区间的宽度 \(h=\frac{b - a}{N}\)。在每个由相邻三个节点 \(x_{i - 1},x_i,x_{i+1}\) 确定的子区间 \([x_{i - 1},x_{i+1}]\) 上，构造一个通过这三个点的二次函数 \(p(x)=Ax^{2}+Bx + C\)，然后计算该二次函数在子区间 \([x_{i - 1},x_{i+1}]\) 上的定积分，以此来近似被积函数 \(f(x)\) 在该子区间上的积分。最后将所有子区间上的近似积分相加，得到整个区间 \([a,b]\) 上的积分近似值。
2. **公式推导**
   - 设积分区间 \([a,b]\) 被分成 \(N\) 个等宽子区间 \([x_0,x_1],[x_1,x_2],\cdots,[x_{N - 1},x_N]\)，其中 \(x_i=a+ih\)，\(i = 0,1,\cdots,N\) 且 \(h=\frac{b - a}{N}\)。
   - 考虑一个子区间 \([x_{i-1},x_{i + 1}]\)（\(i = 1,3,\cdots,N - 1\)，因为 \(N\) 为偶数，这样的子区间成对出现），通过三个点 \((x_{i-1},f(x_{i-1}))\)，\((x_i,f(x_i))\) 和 \((x_{i + 1},f(x_{i+1}))\) 构造二次函数 \(p(x)=Ax^{2}+Bx + C\)。
   - 计算二次函数 \(p(x)\) 在区间 \([x_{i-1},x_{i + 1}]\) 上的定积分：
     - 先将 \(x\) 进行变换，令 \(t=\frac{x - x_i}{h}\)，则 \(x=x_i+th\)，\(dx = hdt\)。当 \(x = x_{i-1}\) 时，\(t=-1\)；当 \(x = x_{i + 1}\) 时，\(t = 1\)。
     - \(p(x)=A(x_i+th)^{2}+B(x_i+th)+C=At^{2}h^{2}+(2Ax_ih + Bh)t+(Ax_i^{2}+Bx_i + C)\)。
     - \(\int_{x_{i-1}}^{x_{i + 1}}p(x)dx=h\int_{-1}^{1}(At^{2}h^{2}+(2Ax_ih + Bh)t+(Ax_i^{2}+Bx_i + C))dt\)。
     - 由于 \(\int_{-1}^{1}tdt = 0\)，\(\int_{-1}^{1}dt=2\)，\(\int_{-1}^{1}t^{2}dt=\frac{2}{3}\)，所以 \(\int_{x_{i-1}}^{x_{i + 1}}p(x)dx=h\left(\frac{2}{3}Ah^{2}+2(Ax_i^{2}+Bx_i + C)\right)\)。
     - 又因为 \(p(x_{i-1})=A(x_{i-1})^{2}+Bx_{i-1}+C\)，\(p(x_i)=Ax_i^{2}+Bx_i + C\)，\(p(x_{i + 1})=A(x_{i + 1})^{2}+Bx_{i + 1}+C\)，经过化简可得 \(\int_{x_{i-1}}^{x_{i + 1}}p(x)dx=\frac{h}{3}[p(x_{i-1}) + 4p(x_i)+p(x_{i + 1})]\)。
   - 对所有子区间 \([x_{i-1},x_{i + 1}]\)（\(i = 1,3,\cdots,N - 1\)）上的积分求和，就得到了 \(\int_{a}^{b}f(x)dx\approx\frac{h}{3}\left[f(x_0)+4\sum_{i = 1,3,\cdots,N - 1}f(x_i)+2\sum_{i = 2,4,\cdots,N - 2}f(x_i)+f(x_N)\right]\)。
3. **适用条件**
   - \(N\) 必须为偶数：因为 Simpson 法则是基于对相邻三个节点确定的子区间（成对出现）构造二次函数进行积分近似的。如果 \(N\) 为奇数，那么在划分区间后，最后一个子区间无法与前面的子区间按照三个节点一组的方式进行处理。
   - 被积函数 \(f(x)\) 在积分区间 \([a,b]\) 上应该是连续的，并且具有一定的光滑性（至少二阶可导），这样才能保证用二次函数近似代替 \(f(x)\) 时具有较好的精度。如果函数在区间内有剧烈的振荡或不连续点，Simpson 法则的近似效果可能会很差。

综上所述，Simpson 法则通过用二次函数近似被积函数来计算定积分的近似值，其公式推导基于二次函数积分和变量代换，适用条件为 \(N\) 为偶数且被积函数在积分区间上连续且具有一定光滑性。 
### 2.2 梯形法则

简述梯形法则的积分思想、公式推导，并给出公式：
$$
\int_a^b f(x)dx \approx h\left[\frac{f(x_0)}{2} + f(x_1) + \cdots + f(x_{N-1}) + \frac{f(x_N)}{2}\right]
$$

以下是梯形法则的相关内容：
- **积分思想**：梯形法则是将积分区间$[a,b]$划分成$N$个等宽的子区间，每个子区间的宽度为$h=\frac{b - a}{N}$。然后在每个子区间上，用梯形来近似代替被积函数与$x$轴之间的曲边梯形面积。具体来说，就是将每个子区间上的函数曲线用连接该子区间两个端点的直线段来近似，这样就形成了一系列梯形，通过计算这些梯形的面积之和来近似计算定积分的值。
- **公式推导**：
    - 对于第$i$个子区间$[x_{i - 1},x_i]$，其梯形面积为$S_i=\frac{1}{2}(f(x_{i - 1}) + f(x_i))h$。
    - 对所有子区间的梯形面积求和，可得：
\[\begin{align*}
\int_a^b f(x)dx&\approx\sum_{i = 1}^{N}S_i\\
&=\sum_{i = 1}^{N}\frac{1}{2}(f(x_{i - 1}) + f(x_i))h\\
&=h\left[\frac{f(x_0)+f(x_1)}{2}+\frac{f(x_1)+f(x_2)}{2}+\cdots+\frac{f(x_{N - 1})+f(x_N)}{2}\right]\\
&=h\left[\frac{f(x_0)}{2}+f(x_1)+f(x_2)+\cdots+f(x_{N - 1})+\frac{f(x_N)}{2}\right]
\end{align*}\]

综上，梯形法则的公式为\(\int_a^b f(x)dx\approx h\left[\frac{f(x_0)}{2}+f(x_1)+f(x_2)+\cdots+f(x_{N - 1})+\frac{f(x_N)}{2}\right]\)。它是一种较为简单的数值积分方法，适用于被积函数在积分区间上变化相对平缓的情况。如果被积函数变化剧烈，可能需要将区间划分得更细以提高精度。

## 三、实验代码

import numpy as np

# 待积分函数
def f(x):
    return x**4 - 2*x + 1

# 已给出的梯形法则积分函数，供参考比较用
def trapezoidal(f, a, b, N):
    h = (b - a) / N
    x = np.linspace(a, b, N+1)
    fx = f(x)
    integral = h * (0.5 * fx[0] + np.sum(fx[1:-1]) + 0.5 * fx[-1])
    return integral

# Simpson 法则积分函数
def simpson(f, a, b, N):
    if N % 2 != 0:
        raise ValueError("Simpson 法则要求 N 必须为偶数")
    h = (b - a) / N
    x = np.linspace(a, b, N+1)
    fx = f(x)
    # 奇数索引（1,3,5,...,N-1）
    odd_sum = np.sum(fx[1:N:2])
    # 偶数索引（2,4,6,...,N-2）
    even_sum = np.sum(fx[2:N:2])
    integral = h / 3 * (fx[0] + 4 * odd_sum + 2 * even_sum + fx[N])
    return integral

def main():
    a, b = 0, 2
    exact_integral = 4.4

    for N in [100, 1000]:
        trapezoidal_result = trapezoidal(f, a, b, N)
        simpson_result = simpson(f, a, b, N)

        trapezoidal_error = abs(trapezoidal_result - exact_integral) / exact_integral
        simpson_error = abs(simpson_result - exact_integral) / exact_integral

        print(f"N = {N}")
        print(f"梯形法则结果: {trapezoidal_result:.8f}, 相对误差: {trapezoidal_error:.2e}")
        print(f"Simpson法则结果: {simpson_result:.8f}, 相对误差: {simpson_error:.2e}")
        print("-" * 40)

if __name__ == '__main__':
    main()

### 4.2 误差分析
- 分析两种方法在不同 $N$ 下的误差表现。
- Simpson 法则与梯形法则误差的数量级差异及原因。
- 讨论 $N$ 增大时误差的变化趋势。

### 两种方法在不同$N$下的误差表现
- **梯形法则**：梯形法则的误差主要是由于用梯形近似曲边梯形时产生的。当$N$较小时，梯形对曲线的近似程度较差，误差较大。随着$N$的增大，每个子区间的宽度变小，梯形对曲线的近似程度提高，误差会逐渐减小。但由于梯形法则只是用直线段来近似曲线，对于一些曲率较大的函数，即使$N$增大，误差的减小也会比较缓慢。
- **Simpson法则**：Simpson法则是用二次函数来近似被积函数，在每个子区间上的近似程度比梯形法则更高。当$N$较小时，Simpson法则的误差就比梯形法则小很多。随着$N$的增大，误差会迅速减小。因为二次函数能够更好地拟合曲线的弯曲程度，所以对于大多数函数，Simpson法则在相同的$N$下能得到更精确的结果。

### Simpson法则与梯形法则误差的数量级差异及原因
- **数量级差异**：一般情况下，梯形法则的误差量级是$O(h^2)$，其中$h=\frac{b - a}{N}$是子区间的宽度。而Simpson法则的误差量级是$O(h^4)$。这意味着在$N$相同的情况下，Simpson法则的误差比梯形法则小得多。例如，当$N$加倍时，梯形法则的误差大约会变为原来的$\frac{1}{4}$，而Simpson法则的误差大约会变为原来的$\frac{1}{16}$。
- **原因**：梯形法则是基于线性近似，它只考虑了函数在子区间端点的值，对函数的局部变化描述能力有限。而Simpson法则采用二次函数近似，不仅考虑了端点值，还考虑了子区间中点的值，能更好地捕捉函数的弯曲特性，从而大大提高了近似精度，使误差的数量级降低。

### $N$增大时误差的变化趋势
- **梯形法则**：随着$N$的增大，$h$逐渐变小，根据误差量级$O(h^2)$，误差会以平方的速度逐渐趋近于零。但如前文所述，对于一些复杂函数，误差减小的速度可能会受到函数特性的限制。
- **Simpson法则**：由于误差量级为$O(h^4)$，当$N$增大时，$h$变小，误差会以更快的速度趋近于零。理论上，只要函数满足一定的光滑性条件，$N$足够大时，Simpson法则可以达到很高的精度。

综上所述，Simpson法则在误差表现上通常优于梯形法则，尤其是在$N$较大时，其误差减小的速度更快，能更准确地逼近定积分的真实值。
## 五、结论
- 总结两种方法的适用场景和优劣。
- 结合本实验结果，说明实际工程中如何选择数值积分方法。
以下是对梯形法则和Simpson法则的适用场景、优劣总结，以及在实际工程中选择数值积分方法的说明：

### 梯形法则
- **适用场景**：适用于对积分精度要求不高，被积函数变化相对平缓，且计算资源有限或计算速度要求较高的情况。例如，在一些初步的估算或对简单函数进行积分时可以使用。
- **优点**：算法简单，易于理解和实现。只需要知道函数在区间端点的值，计算量相对较小，对于一些简单问题可以快速得到一个大致的结果。
- **缺点**：积分精度相对较低，误差量级为$O(h^2)$。对于复杂的、变化剧烈的函数，需要将区间划分得很细才能达到较高的精度，此时计算量会大幅增加。

### Simpson法则
- **适用场景**：适用于对积分精度要求较高，被积函数具有一定的光滑性，且计算资源和时间允许的情况。在科学计算、工程设计等领域中，当需要较为精确的积分结果时，常被采用。
- **优点**：积分精度高，误差量级为$O(h^4)$。通过使用二次函数近似被积函数，能更好地拟合函数的曲线形状，在相同的区间划分下，比梯形法则能得到更准确的结果。
- **缺点**：算法相对复杂一些，需要计算函数在子区间中点的值。并且对于不满足光滑性条件的函数，可能无法发挥其高精度的优势。

### 实际工程中选择数值积分方法的考虑因素
结合上述两种方法的特点以及实验结果，在实际工程中选择数值积分方法时可以考虑以下几点：
- **精度要求**：如果对结果的精度要求较高，例如在一些涉及到精确物理量计算、高精度设计等领域，Simpson法则通常是更好的选择。但如果只是需要一个大致的结果，对精度要求不苛刻，梯形法则可能就足够了。
- **函数特性**：对于光滑的函数，Simpson法则能很好地发挥其优势，获得高精度的结果。而对于不光滑或具有奇异点的函数，可能需要先对函数进行处理，或者考虑使用其他更适合的数值方法，此时梯形法则可能相对更稳健一些。
- **计算资源和时间**：如果计算资源有限或对计算速度要求很高，梯形法则由于其简单快速的特点可能更合适。但如果有足够的计算资源和时间，为了获得更高的精度，Simpson法则是优先考虑的方法。
- **问题的复杂性**：对于复杂的积分问题，可能需要结合多种方法或者进行误差分析来选择最合适的数值积分方法。有时可以先使用梯形法则进行初步计算，再根据需要使用Simpson法则或其他更高级的方法进行细化和验证。

实际工程中选择数值积分方法需要综合考虑精度要求、函数特性、计算资源和时间等多方面因素，根据具体问题的特点来选择最合适的方法，以达到既满足工程需求又能高效计算的目的。
